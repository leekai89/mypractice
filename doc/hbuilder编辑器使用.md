 ##hbuilder使用
 
 ### 几个比较常用的快捷键
 - Ctrl + Enter 光标直接定位到下一行, 无论光标之前是在文字开始中间或者末尾部分,非常实用的一个功能
 - Ctrl + Shift + Enter 光标直接定位到上一行, 和光标定位到下一行命令相反而已
 - Ctrl + Tab 多个文档页面之间来回切换, 不想要鼠标直接点击,是个不错的懒人切换快捷键
 - Ctrl + D 直接删除当前行
